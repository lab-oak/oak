# Installation

The package is not yet on PyPI. Instead, download a wheel for your python minor version from a successful Github Actions run.

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install *.whl
# test
benchmark
```

Instruction for installing and using the package can be found [here](./TUTORIAL.md).
